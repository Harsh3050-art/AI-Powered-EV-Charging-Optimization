# AI-Powered-EV-Charging-Optimization
This project develops an AI-powered EV charging optimization framework using large-scale charging session data to improve demand forecasting, dynamic tariff optimization, and monitoring intelligence.
# Features
- Demand Forecasting
- Dynamic Pricing Optimization
- Monitoring Agent Intelligence
- Spatial & Temporal Analysis
- Revenue Optimization
- Queue Reduction Analysis
# Project Flow
- 01 data_preprocessing.ipynb
- 02 feature-processing.ipynb
- 03 temporal-eda.ipynb
- 04 spatial analysis.ipynb
- 05 acn-behavioral-analysis(1).ipynb
- 06 demand-prediction-agent.ipynb
- 07 tariff-pricing-agent.ipynb
- 08 monitoring-learning-agent.ipynb
